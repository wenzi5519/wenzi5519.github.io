几个常用的命令：
1.hexo g(hexo generate) 生成静态文件，会在当前目录下生成一个叫做public的文件夹
2.hexo s(hexo server) 启动本地服务，用来预览博客
3.hexo d(hexo deploy) 部署到远端(比如github)
4.hexo n “new”(hexo new “new”) 新建文章
